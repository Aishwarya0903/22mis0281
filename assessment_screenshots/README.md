# Assessment Screenshots

This folder is for the API client screenshots required by the spec:

> Capture output screenshots from API clients like Insomnia or
> Postman, displaying request body, response and response time for
> the problem.

Recommended naming so a reviewer can navigate at a glance:

```
vehicle_scheduler_GET_schedule.png
vehicle_scheduler_GET_schedule_dryrun.png
vehicle_scheduler_GET_health.png

notifications_GET_top.png
notifications_GET_top_explain.png

logger_POST_logs_success.png
logger_POST_logs_validation_error.png

auth_POST_auth_token.png
```

Each screenshot should show, at minimum:

- The full request URL and method
- Request body (where applicable)
- The response payload
- The response time

Both Insomnia and Postman display all four by default. Bruno also
works if you prefer an offline client.
